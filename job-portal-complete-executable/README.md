# Job Portal Complete Executable

## Steps
1. Open MySQL Workbench.
2. Run `database.sql` completely.
3. Open this folder in IntelliJ.
4. Open `src/main/resources/application.properties` and replace:
   `CHANGE_YOUR_MYSQL_PASSWORD_HERE` with your MySQL password.
5. Run `JobportalApplication.java`.
6. Open: `http://localhost:8080/index.html`
